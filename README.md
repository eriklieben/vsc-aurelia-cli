# Visual Studio Code plugin for Aurelia-CLI

## Install

In Visual Studio Code, run ```ext install vsc-aurelia-cli``` - or, 
simply hit CMD/CTRL + Shift + P , search for "Install Extension", 
and then search for "aurelia-cli".

## Supported commands

- au build
- au generate
- au new
- au test
